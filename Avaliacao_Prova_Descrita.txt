Questões:

1 - Qual a utilidade do arquivo hibernate.cfg.xml? Ele é dispensável para o funcionamento do hibernate? Que tipo de informação ele contém?
	Ele é o arquivo de configurações para o funcionamento do hibernate é nele que se faz o mapeamento das classes para a construção do Banco! Ele é indespensável pois
	contem as configurações necessarias para o funcionamento do hibernate!

2 - Qual a utilidade da classe HibernateUtil.java?  Ela é dispensável para o funcionamento do hibernate?
	Para auxiliar na criação das sessoes, para que haja as transações! Ela é indispensavel!

3 - Quais as vantagens de se utilizar Hibernate em detrimento de um acesso ao banco usando ResultSet.
	Mais agilidade em desenvolver o codigo!
	

4 - O que significam cada uma das anotações a seguir:
a) @Entity = Para replicar a classe no banco de dados
b) @Column = Para nomear a coluna com um nome diferente do atributo da classe java
c) @Id = Destaca a chave primaria no Banco
e) @GeneratedValue = Faz com que o atributo seja auto incrementado no banco
